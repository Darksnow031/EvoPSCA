        # EvoPSCA

        Reproducibility package for the EvoPSCA method (RL + CEM hybrid optimization).

        ## Requirements

        - Python 3.11
        - Dependencies listed in `requirements.txt`

        ## Run

        After extracting this package:

```bash
pip install -r EvoPSCA_release/requirements.txt
cd EvoPSCA_release/MUJOCO
python -m modules.compat.run_EvoPSCA
```

        Results are saved under `EvoPSCA_release/runs/`.

        ## Package contents

        This folder is self-contained. No files outside `EvoPSCA_release/` are required.
